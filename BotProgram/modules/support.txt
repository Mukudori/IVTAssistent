	Программа предназначена для управления бот-мессседжером.

Используемая база данных: MySQL;
Язык программирования: Python
Версия интерпретатора: Python 3.6

Из программы присутствует возможность редактирования БД бота и его тестированиe.

В режиме тестирования бот может:
- слушать команды через микрофон;
- озвучивать ответы на вопросы; 
- выполнять расширенный набор команд администратора;
- запускаться в Telegram.
